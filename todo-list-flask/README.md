# To-Do App 

A simple and clean To-Do List web application built using Flask and SQLite. This project allows users to manage daily tasks efficiently with a minimal and user-friendly interface.

---

## Features

* Add new tasks with due date
* Mark tasks as completed
* Delete tasks
* Color indication for task status (Completed / Pending)
* Simple and clean UI using HTML & CSS

---

## Tech Stack

* Python (Flask)
* SQLite
* HTML
* CSS

---

## How to Run

1. Install Flask

```
pip install flask
```

2. Run the application

```
python app.py
```

3. Open in browser

```
http://127.0.0.1:5000/
```
